## 📌 Overview  
This project is an interactive machine learning application that predicts California housing prices based on user inputs. The model is deployed using Streamlit Cloud, allowing users to adjust key housing and demographic features and receive real-time predictions.

---

## 🎯 Objective  
To build and deploy a predictive model that estimates housing prices using socioeconomic and geographic features, demonstrating an end-to-end machine learning workflow from training to deployment.

---

## ⚙️ Project Structure  

| File | Description |
|------|------------|
| `app.py` | Streamlit application |
| `model.pkl` | Trained machine learning model |
| `feature_names.pkl` | Feature names for input consistency |
| `requirements.txt` | Project dependencies |
| `README.md` | Project documentation |
